Quick Start

Up and running
in 3 minutes.

Install once globally. Set up your tokens once. Convert forever.

Step 01

Install

Install f2c globally from npm. Requires Node.js 18+.

npm install -g f2c-cli
Step 02

Configure

Run the setup wizard. You'll need a Figma access token and an OpenAI / Claude API key.

f2c init
Step 03

Convert

Copy any Figma node URL and convert it to React or Vue code.

f2c convert -u "<figma-url>" -f react

FAQ

Common questions.

Everything else, email me at duziteng1987@gmail.com

What AI provider does f2c use?

f2c supports OpenAI (GPT-4o, GPT-4-turbo) and Anthropic (Claude 3). You supply your own API key — f2c never stores it. Run f2c init to configure.

Where do I get a Figma access token?

Open Figma → Settings → Security → Personal access tokens → Generate new token. Paste it when you run f2c init. Read-only access is enough.

How quickly does the license key arrive?

Usually within 1 minute of purchase. It's sent automatically to the email you use at checkout. Check your spam folder if it doesn't appear.

Can I use the key on multiple machines?

Yes. Run f2c license activate <key> on each machine you use. The license is not locked to a single device.

What if I don't receive my key?

Email duziteng1987@gmail.com with your Gumroad order number and I'll send the key manually, usually within a few hours.

Is there a refund policy?

Yes — 7-day no-questions-asked refund. Just email with your order number. Refunds are processed through Gumroad and the key will be deactivated.

Does f2c work offline?

The f2c convert command requires internet access (Figma API + AI). The license is cached locally for 7 days, so Pro features work offline after first activation.

What Node.js version is required?

Node.js 18 or higher. Run node -v to check. Most systems already meet this requirement.
